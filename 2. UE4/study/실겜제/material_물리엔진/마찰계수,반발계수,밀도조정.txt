1. 추가/임포트 : 피직스-> 피지컬 머터리얼
2. 적용시킬 메시에 해당 피지컬 머터리얼 적용. (우리의 경우 box)
3. 적용 후 피지컬 머터리얼 더블 클릭 후 friction 0.3, override friction combine mode 체크, friction combine mode min으로 변경
4.  그후 반발 계수 restitution 을 0.1, 밀도를 0.3 


반발계수는 물체의 충돌 전후 속도의 비율을 나타내는 것으로, 반발계수가 1인 물체는 탄성 충돌을 하고, 반발계수가 1보다 작은 물체는
비탄성 충돌을 한다.
0이면 완전 비탄성 충돌을 하며, 충돌한 물체와 붙어서 튀지 않음.


마찰 계수 (Friction)

1에 가까울 수록 마찰이 커지고, 강체를 움직이기 어려움. 움직이다가도 쉽게 멈춤
0에 가까울 수록 마찰이 약해지고, 강체를 움직이기 쉬우며 한번 움직이면 잘 안멈춤


마찰 합성 모드 (Friction Combine Mode)
마찰력은 2개의 강체가 접촉할때 발생하는 힘.

average - 디폴트로, 2개의 강체가 가진 friction 평균값을 사용
multiply - 두개의 강체가 가진 friction 값을 곱해서 사용 
마찰은 0~1 범위로 설정하므로, 곱하면 값이 작아진다.
따라서 기본적으로 곱하면 마찰이 약해짐.
1을 넘는 마찰 설정 또는 0 으로 설정된 존이 있는 것처럼 특수한 마찰 설정으로 게임 세계를 만들때 사용

max - 2개의 강체가 가진 friction 의 값 중에 큰 값을 사용 
최대한 물체를 움직이게 하지 않을 때 사용. ex) 얼음 위에서 너무 많이 미끄러지지 않게 만들때 사용

min - 2개의 강체가 가진 friction 값 중 작은 값을 사용.
최대한 물체를 움직이고 싶을때, 멈추면 안되는 경우 사용 ex) 모래 사막에서 공을 굴릴떄 




density(밀도)
마찰력은 마찰계수 * 질량
